宇树科技 / 王兴兴 亚布力论坛 BREAKING
王兴兴亚布力开幕演讲：距具身智能的"ChatGPT 时刻"还需 2-3 年，年中机器人将跑赢博尔特
2026 亚布力中国企业家论坛年会（3/17-19）在黑龙江亚布力开幕，宇树科技创始人、CEO 王兴兴 以论坛新一届理事身份发表开幕演讲，引发广泛关注。

王兴兴首次公开定义了"具身智能 ChatGPT 时刻"——即在 80% 的陌生场景中，通过语音或文字指令，机器人能够顺利完成大约 80% 的任务。他表示，目前距离这一时刻"可能至少还要 2-3 年时间——但也可能非常快"。

更引人注目的是他对短期目标的大胆预测：预计 2026 年中，全球特别是中国的人形机器人将在百米跑中跑得比人还快，"比博尔特更快一些"。这一判断呼应了前一天 GTC 上 NVIDIA Isaac GR00T 机器人平台的发布，中美两大会场在机器人议题上形成共振。

新浪财经 / 行业分析 新浪财经 INSIGHT
"养虾"之后，AI Agent 如何跃迁？——深度分析 OpenClaw 从开源项目到操作系统级平台的演化
新浪财经发布深度长文，系统梳理了 OpenClaw 从一个极客色彩的开源项目到现象级 AI Agent 平台的演化路径。文章核心观点：

1) OpenClaw 点燃的不只是流量——它代表了一种新预期：AI 已从"会说"进入"会做"的阶段。真正值得追问的不是 OpenClaw 火不火，而是它代表的 Agent 时代是否真的到来；
2) 三个演化阶段——第一阶段"养虾"式娱乐体验 → 第二阶段专业化工具链（代码、数据分析、自动化运维）→ 第三阶段 Agent 操作系统（NemoClaw + Vera Rubin 的硬件底座）；
3) 核心瓶颈——安全管理（中国人工智能产业联盟已发布风险管理指南）、长时间任务的可靠性、以及 Agent 间协作标准的缺失。

文章结论认为，OpenClaw 是 Agent 时代的"iPhone 时刻"前奏，但距离真正的操作系统级平台还有 1-2 年。
